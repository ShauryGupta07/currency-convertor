# React Currency Converter

A responsive currency converter built with React and Tailwind CSS that fetches real-time exchange rates using a public currency API. Users can convert between multiple currencies, swap currencies instantly, and view accurate conversion results through a clean and intuitive interface.

## 📸 Preview

![Project Preview](./screenshots/preview.png)

## ✨ Features

* 💱 Convert between multiple international currencies
* 🌐 Fetch real-time exchange rates using a public API
* 🔄 One-click currency swap
* 📋 Dynamic currency selection
* ⚡ Fast and responsive user interface
* 📱 Mobile-friendly design

## 🛠️ Tech Stack

* React
* JavaScript (ES6+)
* Tailwind CSS
* Vite
* Currency Exchange API

## 📚 React Concepts Used

* `useState` – Manage application state
* `useEffect` – Fetch exchange rates when the base currency changes
* `useId` – Generate unique IDs for accessible form elements
* Custom Hooks – Reusable API fetching logic (`useCurrencyInfo`)
* Component Reusability – Reusable `InputBox` component
* Props – Pass data and callback functions between components
* Controlled Components – Manage form inputs using React state

## 🚀 Installation

1. Clone the repository

```bash
git clone <https://github.com/ShauryGupta07/currency-convertor.git>
```

2. Navigate to the project directory

```bash
cd <repository-name>
```

3. Install dependencies

```bash
npm install
```

4. Start the development server

```bash
npm run dev
```
